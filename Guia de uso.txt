passo 1: abrir o menu "Extensões"

passo 2: pesquisar "Python"

passo 3: clicar em instalar na extenção "Python"

passo 4: voltar para o menu "Explorador"

passo 5: usar o comando "python3 -m pip install --upgrade pip" no terminal para atualizar o instalador

passo 6: usar o comando "pip install -r requirements.txt" no terminal para baixar todas as bibliotecas necessárias

passo 7: colocar o arquivo "credentials.json" na pasta "Data"

passo 8: editar a planilha de clientes

passo 9: entrar no arquivo "main.py"

passo 10: mudar o valor da variável quantidade_de_caminhões para a quantidade de caminhões desejada

passo 11: clicar no play no canto superior direito do navegador

passo 12: clicar com o botão direito do mouse em um dos arquivos pdf gerados com o nome "roteiro_entregas_caminhão(numero do caminhão).pdf"

passo 13: clicar em baixar no menu suspenso

passo 14: clicar com o botão direito do mouse em um dos arquivos xlsx gerados com o nome "roteiro_entregas_caminhão(numero do caminhão).xlsx"

passo 15: clicar em baixar no menu suspenso

passo 16: abrir os arquivos

passo 17: conferir se deu certo
